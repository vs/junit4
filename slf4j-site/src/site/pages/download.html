Maven central

SLF4J artifacts such as .jar, .sources.jar and javadoc.jar files can be downloaded from Maven central. The corresponding groupId is org.slf4j.

Verifying contents

All SLF4J artifacts published on Maven central are signed. For each artifact, there is an associated signature file with the .asc suffix.
